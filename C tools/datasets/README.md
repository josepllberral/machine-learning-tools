## Datasets

Here you can find the datasets for **MNIST** and **MOTION** (fragment).

**MNIST** binarized data files
* mnist_testx.data
* mnist_testy.data
* mnist_trainx.data
* mnist_trainy.data

**MOTION** (fragment) binarized data files
* motion_bd.data
* motion_dm.data
* motion_ds.data
* motion_sl.data

*Note*: The data files must be unzipped (please, don't upload them to the repo)
